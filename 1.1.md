# 1.1 - Basics

## Display variables

TypeScript files can be displayed in HTML by being surrounded by a double curly brackets:

The value of `variable` is **3**.

``` html
<span>{ { variable } }</span>
```
> display **3**

``` html
<span>1 + 1 = { { variable } }</span>
```
> display **1 + 1 = 3**

``` html
<span>{ { 1 + 1 + variable } }</span>
```
> display **5**
